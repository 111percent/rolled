
# rolled
[![npm version](https://badge.fury.io/js/rolled.svg)](https://badge.fury.io/js/rolled)

rolled is a Javascript library for 
- dom renderer 
- Ownership hooks
- pure
- virtual Component (not "only dom")

## Installation
```bash
npm install rolled
```
